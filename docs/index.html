About

Display web maps in any CRS available in PROJ, using proj-wasm for in-browser projection math.

MapLibre integration via maplibre-proj:

const { style, bounds } = await reprojectStyle({
  style, crs: 'EPSG:5070',
});
map.setStyle(style);
map.fitBounds(bounds);
